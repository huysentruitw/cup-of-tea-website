Over

Hoi, ik ben Wouter, een creatieve duizendpoot met een uitgesproken interesse voor alles wat digitalisering aangaat. Als software architect en doorwinterde software ontwikkelaar bied ik onder de naam Cup of Tea mijn diensten aan.

Gedurende de voorbije 20 jaar had ik het voorrecht met heel veel verschillende technologieën in contact te komen. Zo heb ik ervaring met het ontwikkelen van hardware, firmware, stuurprogramma's, desktop- en webapplicaties. Op vandaag ligt mijn specialisatie voornamelijk in Microsoft .NET technologieën, waarvoor ik ook de Microsoft MVP award mocht ontvangen.

Een beknopt overzicht van mijn vaardigheden:

  • .NET Core / C#
  • Angular / TypeScript
  • Clean code
  • REST / GraphQL
  • Test-driven development (TDD)
  • EntityFramework Core / Dapper
  • Microsoft SQL
  • Oracle
  • Windows Server
  • Azure DevOps (CI/CD)

Diensten

Maatsoftware

De maatsoftware wordt steeds gebouwd op basis van moderne webtechnologieën om ervoor te zorgen dat deze op verschillende toestellen werkt (denk maar aan smartphone, tablet, desktop).

Om dit te verwezenlijken gebruiken we technologieën als .NET Core, EntityFramework Core, GraphQL, IdentityServer, Angular, Azure DevOps en meer.

De software is klaar voor de toekomst en heel eenvoudig aanpasbaar door het toepassen van bewezen methoden en technieken.

Consultancy

Nood aan wat extra mankracht of hulp nodig bij het oplossen van complexe problemen waar je huidig team van developers op vast zit?

Problemen worden een uitdaging waar we onze tanden in vastzetten tot we samen de best passende oplossing hebben gevonden.

We komen ook graag spreken binnen je team over alles wat met programmeren te maken heeft. Zoals het werken met git, GitFlow, Semantic Versioning, CI/CD met Azure DevOps, Clean Code, Test-driven development, CQRS + EventSourcing, ...

Webdesign

Ondanks de ontelbare kanalen waar je als bedrijf reeds actief bent, blijft een eigen stek op het web onontbeerlijk.

We verzorgen uw website tot in de puntjes en zorgen er ook voor dat deze gemakkelijk gevonden wordt door SEO optimalisatie.

Bij Cup of Tea hebben we meer dan 15 jaar ervaring met het maken van mooie en toegankelijke websites.
